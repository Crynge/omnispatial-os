# Architecture

## Verified Core

The verified core combines a Python spatial intelligence engine, a FastAPI control plane, and a React dashboard.

### Core layers

1. **Spatial signal fusion**
   - foot traffic
   - weather pressure
   - event intensity
   - competitor activity
   - WiFi occupancy estimates
2. **Agent swarm**
   - trend agent
   - creative agent
   - budget agent
   - operations agent
3. **Causal what-if layer**
   - zone-level ROI uplift
   - offline-to-online spillover estimates
   - sensitivity-aware scenario narration
4. **Command center**
   - hyperlocal zone scoring
   - agent recommendations
   - signal health and confidence surfaces

## Polyglot Expansion Surfaces

The broader platform is structured for future specialization across:

- **Rust** spatial signal processing and orchestration
- **Go** edge gateway services
- **Scala** historical batch processing
- **C++ / WebAssembly** edge inference
- **Swift / Kotlin** field marketer approval flows
- **SQL / PostGIS-style queries** for geospatial targeting and store intelligence

The verified runtime stays compact and testable while the repository communicates a realistic long-term system design.
